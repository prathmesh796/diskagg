
# Disggle

A discord presence app for showing off your kaggle status.


## Installation

### Step 1
Download the /extenstion/dist/ folder and unpack it in your chrome extenstions. Before that make sure your are in developer mode.

### Step 2
Add this discord app to your account:

[discord app](https://discord.com/oauth2/authorize?client_id=1437065833137438754)

Also make sure that you are logged into your discord account in your browser.

### Step 3
Download the /rpc-server/ folder and run following cmds in that folder:

```
npm install
npm run start
```
